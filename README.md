## Address Book 📕

## Demo online 🕹️

Check it online 👉 [here](https://elated-kilby-fa4dbb.netlify.app/)

## Rodar ⚙️
clone o repositório e rode o comando
```yarn```
### Para iniciar o servidor
```yarn start```
