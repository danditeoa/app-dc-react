## Rodar ⚙️
clone o repositório e rode o comando
```yarn```
### Para iniciar o servidor
```yarn start```
### Using Docker 🐋

On root folder:

```docker build -t addressbook .```
```docker run -p 80:80 -d addressbook```

The app will be running on [localhost](http://localhost/)
